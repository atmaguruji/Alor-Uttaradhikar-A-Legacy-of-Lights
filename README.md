## About

This repository hosts the complete multimedia production of **"Alor Uttaradhikar – A Legacy of Lights"**, a cinematic Bengali audio documentary drama commemorating the **125th Birth Anniversary of Dr. Shyama Prasad Mukherjee**.

The project integrates literature, theatre, narration, music, historical research, education and digital publishing into a single multimedia presentation.
